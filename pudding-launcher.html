Pudding Launcher

Info

The pudding launcher is my personal launcher in which I distribute my modpacks. These are, for example, modpacks for my modded servers or just some to improve the vanilla gaming experience.

Features

The Launcher

The launcher is based on the SKCraft Launcher and is an easy way to provide my modpacks.

Account Manager

The launcher is able to save multiple accounts, regardless if Mojang or Microsoft. Alternatively, you can play without an account.

Selectable Features

Another feature is that optional mods can be selected when installing a modpack.

Download

Note that the launcher requires at least Java 8 and the latest versions of Minecraft Java 16!
