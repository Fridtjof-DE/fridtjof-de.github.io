My Projects

This is a list of some of my projects.

Moto-Wiki

Diese Seite dient dem Zusammentragen von Informationen und Anleitung zu einigen alten Mopeds und Motorrädern. Dabei wird auch Wert auf Kleinigkeiten gelegt, die sonst eher nicht erwähnt werden.

Pudding Launcher

The pudding launcher is my personal launcher in which I distribute my modpacks. These are, for example, modpacks for my modded servers or just some to improve the vanilla gaming experience.

Missing_Colors

A simple Bukkit plugin which allows you to use formatting codes ingame. It is compatible with over 50 versions of Minecraft.

Ore Grids

A texture-pack for Minecraft that adds an outline to ores in order to recognize them better. It supports 13 mods and over 150 ores.

Sparkle Client

A discontinued Minecraft-PvP-client which features cosmetics, draggable HUD elements, item physics and other mods.
