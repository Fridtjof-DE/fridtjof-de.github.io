My Projects

This is a list of some of my projects.

Sparkle Client

A currently work in progress Minecraft-PvP-client which features things like cosmetics, draggable HUD elements, item physics and other mods.

Missing_Colors

A simple Bukkit plugin which allows you to use formatting codes ingame. It also features an update-checker and is compatible with ~50 versions of Minecraft.

Ore Grids

A texture-pack for Minecraft that adds an outline to ores in order to recognize them better. It supports 13 mods and over 150 ores.
